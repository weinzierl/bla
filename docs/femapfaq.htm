What does "Preprozessor control Validation failed." mean?

 The reason is most probably special characters in the input file. When the file checksum is calculated FEMAP and NASTRAN treat special characters differently, and this is where the error comes from. This also applies to entity names, because these are written as comments into the DAT file and comments are part of the checksum calculation.

What does "Error Code 9002" mean?

Spaces in the model name, spaces in the path to the model file, spaces in the path to FEMAP. Special characters, like Ã¤are also forbidden in path and file name.

How can I make FEMAP output the max and min values of front- and backside of plate lements combined?

In Model/Output/Envelope chose From Output Set, and select the appropriate Output Set. As Vector select Plate Top und Plate Bot. A new Output Set with min and max values will be generated.

How can I get information about a certain node or element quickly?

On the right side of the status bar select the Entity you would want to show. Move you mouse pointer over the element you want to query and leave it there. A small window with selected info is shown. Which info is shown depends on the selected Output Set and the selected Output Vector. This functionality is refered as Dynamic Query in the manual. Other useful functions are: Dynamic Query + left mouse button copies text in the List Window Dynamic Query + right mouse button copies text in the Graphic Window To view the test, it must be in the current Group and a Redraw has to be done.
